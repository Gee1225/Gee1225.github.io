📊 Tweet Sentiment Visualization App

A data visualization web app built with Streamlit that analyzes tweet sentiment data from airline customers. Users can explore patterns by sentiment, airline, hour of the day, and day of the week, with interactive maps and charts.

🚀 Launch Visualization App

Technologies Used:

  • Python & Streamlit
  • pandas & numpy
  • plotly & matplotlib
  • wordcloud

Description:

This app allows users to:

  • Explore sentiment distribution by airline (e.g., United, Delta, Virgin America)
  • Visualize tweet activity across different hours of the day using maps
  • Analyze how sentiment varies across days of the week
  • Generate random tweet samples for each sentiment
Built to demonstrate interactive storytelling and data exploration in real-time with a polished UI.
